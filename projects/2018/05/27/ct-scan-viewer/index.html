CT Scan Viewer

  • Source Code

Created for a university graphical programming unit to carry out manipulations and to display an image from a CT Scan. Written in Java using JavaFx for the UI.

Project Summary: Application to view CT scan data to aid medical professionals.

  • Java
  • JavaFX

Notable Features

  • View 3D and manipulate 3D point data – Top/Side/Front sliced views.
  • Applies Histogram equalization - All data making up the models are saved to a corrisponding csv file.
  • Maximum Intensity Projection (MIP)
